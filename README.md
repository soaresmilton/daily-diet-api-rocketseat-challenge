# daily-diet-api-rocketseat-challenge


Instruções

Estrutura, regras e requisitos do projeto

Seu desafio é criar uma API RESTful para controle de dieta diária utilizando Flask, onde será possível gerenciar refeições de forma completa e organizada. O objetivo é entregar um sistema funcional que permita aos usuários acompanhar sua alimentação diária, registrando detalhes importantes sobre cada refeição e mantendo controle sobre sua dieta.

Como se trata de um desafio avançado, você precisará ter autonomia para pesquisar e estruturar as rotas e propriedades dos registros, exercitando suas habilidades de desenvolvimento e arquitetura de APIs.


[] Implementar registro de refeição com nome, descrição, data/hora e indicador dentro da dieta
[] Desenvolver funcionalidade para editar uma refeição alterando todos os dados
[] Criar funcionalidade para apagar uma refeição
[] Implementar listagem de todas as refeições de um usuário
[] Desenvolver visualização de uma única refeição
[] Configurar salvamento das informações em banco de dados
